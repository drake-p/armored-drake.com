Fleur-de-Lys Inlay

This one follows the same pattern as the inlay on my Fleur hauberk.  I made it as a prototype for that larger project, to be sure it would look right.  You'll notice I finally got my ring size more or less right, but only because I wanted to have the same AR as I was using on the hauberk.  I knew armor was supposed to be dense.
